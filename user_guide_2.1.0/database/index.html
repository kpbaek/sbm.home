데이터베이스The Database Class

CodeIgniter는 완벽하고 빠른 데이터베이스 클래스를제공합니다. 또한 그 그클래스들은 기존의 구조와 액티브레코드Active Record)를 동시에 지원합니다. 데이터베이스 함수들은 깨끗하고 단순한 문법으로 되어있습니다..

  • 시작하기 : 사용예제 (번역)
  • 데이터베이스 설정 (번역)
  • 데이터베이스 연결 (번역)
  • 쿼리 실행(번역)
  • 쿼리 결과 생성 (Generating Query Results)(번역)
  • 쿼리 헬퍼 함수들(Query Helper Functions)(번역)
  • 액티브레코드(Active Record Class)(번역)
  • 트랜젝션 (번역)
    • 테이블 메타테이터(MetaData)(번역)
  • 필드 메타데이터(MetaData)(번역)
  • 사용자 정의 함수 호출(Custom Function Calls)(번역)
  • 쿼리 캐시(Query Caching)(번역)
  • 데이터베이스 관리(Forge)를 이용한 데이터베이스 조작(번역)
  • 데이터베이스 유틸리티 클래스 (번역)
