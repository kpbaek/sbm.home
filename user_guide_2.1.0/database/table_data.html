테이블 메타테이터Table Data

이함수는 테이블에대한 정보를 가져옵니다.

$this->db->list_tables();

이 함수는 현재 연결된 데이터베이스의 모든 테이블 이름을 배열로 가지고 옵니다:

$tables = $this->db->list_tables();

foreach ($tables as $table)
{
   echo $table;
}

$this->db->table_exists();

이 함수는 쿼리를 실행하기전에 해당테이블이 존재하는지 확인하는데 유용합니다 . 호출결과는 TRUE 혹은 FALSE 입니다. 사용예:

if ($this->db->table_exists('table_name'))
{
   // some code...
}

Note: table_name 부분을 여러분이 실재 조회하고자하는 테이블 이름으로 바꾸어주세요
