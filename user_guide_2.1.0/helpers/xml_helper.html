XML Helper

XML 헬퍼는 XML 데이터를 다루는데 필요한 함수를 제공합니다.

헬퍼로딩 Loading this Helper

다음과같이 헬퍼를 로드합니다:

$this->load->helper('xml');

다음 함수들이 사용가능 합니다:

xml_convert('string')

문자열을 입력받아 다음의 예약된 XML 문자들을 엔티티(entities)로 변환합니다:

엠퍼센드: &
보다작음,보다큼 문자들: < >
싱글,더블 인용부호: '  "
대쉬: -

엔티티의 일부로서 존재하는 엠퍼센드는 무시합니다. 예 :

$string = xml_convert($string);
