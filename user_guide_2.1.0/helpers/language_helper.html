언어 Language Helper

언어헬퍼는 언어파일들을 조작할때 도움이 되는 함수들을 제공합니다.

헬퍼로딩 Loading this Helper

다음과같이 헬퍼를 로드합니다:

$this->load->helper('language');

다음 함수들이 사용가능 합니다:

lang('language line', 'element id')

이함수는 로딩된 언어파일로 부터 한줄의 텍스트를 리턴합니다. $this->lang->line() 함수를 호출하는것보다 문법적으로 더 간단하기때문에 이함수를 사용하는것이 바람직합니다 . 옵션인 두번재 파라미터는 출력에 label 을 붙여줍니다. 예:

echo lang('language_key', 'form_item_id');
// becomes <label for="form_item_id">language_key</label>
