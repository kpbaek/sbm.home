이메일 Email Helper

이메일 헬퍼는 이메일관련 작업을 도와줍니다. 보다 강력한 솔루션은 이메일클래스(Email Class) 를 참고하세요.

Loading this Helper

다음과같이 헬퍼를 로드합니다:

$this->load->helper('email');

다음 함수들이 사용가능 합니다:

valid_email('email')

이메일 주소의 적합성 여부를 검사합니다. 주의하실것은 이메일이 실재로 메일을 수신할수 있는지 여부를 검사하는것이 아니라, 주소포멧이 유효한것인지만 검사한다는 점입니다.

이 함수는 TRUE/FALSE를 리턴합니다.

$this->load->helper('email');

if (valid_email('email@somesite.com'))
{
    echo 'email is valid';
}
else
{
    echo 'email is not valid';
}

send_email('수신자', '제목', '내용')

PHP에 내장된 mail() 함수를 이용하여 메일을 보냅니다.
